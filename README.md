A command-line Magic the Gathering collection manager tool.

> [!NOTE]
> Spellbook is still in development and not ready for actual use. Some things may be broken.

# How to Install

If you have Just installed already you can simply:
- Clone this repository
- Navigate to the repository from the command line
- Do `just install`

To install without Just:
- Clone this repository
- Navigate to the repository from the command line
- Install requirements `pip install -r requirements.txt`
- Build the package `python -m build`
- Install the package `python3 -m pip install ./dist/*.whl`

# How to Use

You can show all the commands with `spellbook --help`.
